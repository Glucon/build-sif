# Build SIF Files

A collection of scripts to build Singularity Image Format (SIF) files for various applications.

## Included Build Scripts

```bash
g09/
g16/
orca601/
orca610/
vasp642/
vasp642-intel/
lammps22Jul2025/
lammps-apt/
molpro202420/
cp2k-apt/
psi4-191/
```

## Usage

Navigate to the desired application directory:

```bash
cd [directory_name]
```

Run the build script (need `sudo` or `root`):

```bash
./build.sh
```
